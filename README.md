## 概述

一组 Tampermonkey 用户脚本学习样例，围绕深圳大学（SZU）和华中农业大学（HZAU）体育场馆预约页面的前端注入、表单配置、定时流程、请求控制和 UI 交互进行技术研究。本文档仅说明代码结构和交互设计，不鼓励、不建议、也不授权用于实际预约。

> **免责声明**：本项目仅供技术学习、脚本开发交流与自动化流程研究使用，不可用于盈利、代抢、实际预约、绕过平台规则或任何违反学校/平台规定的用途。使用者应自行承担因不当使用产生的全部后果。

👉 **SZU 学习说明页**：<https://www.280468.xyz/SZU_Sports/>

## ✨ 技术点概览（通用）

- **表单配置**：多时段、场馆策略和参数配置的前端状态管理示例。
- **定时流程**：定时触发、倒计时显示和状态恢复逻辑示例。
- **请求控制**：重试、限频与并发控制的代码结构示例。
- **多端 UI**：移动端触控优化、桌面端悬浮面板控制。
- **轻量 UI**：支持明暗主题、快捷设置浮层、固定日志框与优雅过渡动画。
- **通知示例**：企业微信 Webhook 调用结构示例。

## SZU 版（深圳大学）

- **核心脚本**：见 [src/SZU_Sports.js](src/SZU_Sports.js)。
- **学习说明**：访问 <https://www.280468.xyz/SZU_Sports/> 查看相关页面说明。

### 版本更新（v1.2.7）

- 重构 SZU 面板 UI，加入明暗主题、图标化控件和更克制的视觉样式。
- 新增摘要区快捷设置：姓名/学号、日期、项目、校区、预约场馆、时间段、请求参数均可直接修改。
- 将完整设置、快捷设置、预约小贴士改为独立浮层，不挤压主界面内容。
- 优化动画过渡、滚动条风格、固定日志框高度和进度显示格式。
- 将“优先场馆”文案调整为“预约场馆”，与实际只预约所选场馆的逻辑一致。

### SZU 学习提示

- 代码会在匹配页面注入悬浮按钮和配置面板，可用于学习用户脚本 UI 注入方式。
- 摘要区可用于学习和演示日期、项目、校区、预约场馆和时间段等配置交互。
- 选择具体“预约场馆”时，脚本逻辑会限定该场馆；选择“全部场馆”时才不限定场馆。
- 请求间隔、最大重试、请求超时可在摘要底部快捷修改，用于理解限频和重试逻辑。
