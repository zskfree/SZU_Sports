    2. 查看 电脑端图文教程
    3. 打开下方预约入口即可开始
  • 手机端
    1. 安装 Via 浏览器 (极致轻量)
    2. 查看 手机端图文教程
    3. 打开下方预约入口即可开始

核心功能
  • [通用] 定时抢票 + 自动重试 + 面板可视化
  • [通用] 企业微信推送（可选）+ Token 状态提醒
  • [SZU] 多场馆支持，适配校内/校外访问
  • [HZAU] 盲打预加载 + 服务器校时（可选），提升开抢命中率
支持：深圳大学（SZU） / 华中农业大学（HZAU）
